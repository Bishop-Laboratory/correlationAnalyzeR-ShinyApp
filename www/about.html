Biological insights from gene correlation data

Background

Recent initiatives have re-processed and standardized publicly available RNA-Sequencing data, providing the capability to simultaneously mine thousands of studies. Utilizing the ARCHS4 data repository, this study obtained standardized gene counts from 180,000 sequencing samples, parsed them into tissue groups, and derived genome-wide co-expression correlations. Correlation AnalyzeR was built to allow users to analyze these correlations and extract the wealth of biological insight they can provide. Three analysis modes are available to users:

  1. Single mode
  2. Paired mode
  3. Topology mode

Paired mode


Use paired mode to reveal the location of secondary genes within a primary gene's correlation distribution:


Permutation testing determines whether secondary genes are significantly correlated compared to random chance:


Detail: A list of random genes identical in size to the secondary gene list is selected and a t test compares their correlation values against those of the secondary gene list. This is repeated with 2000 random permutations. The peak p value density represents the most likely true significance of the secondary gene list vs random.
